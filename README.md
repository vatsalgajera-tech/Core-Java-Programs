# Core Java Advanced Programs

This repository contains Core Java programs demonstrating advanced Java concepts such as Multithreading, File Handling, Serialization, Deserialization, and Socket Programming.

## Programs Included

### Multithreading

* ThreadDemo.java
* SynchronizationDemo.java
* UserDefinedExample.java

### File Handling

* ByteStreamReadWrite.java
* CharStreamReadWrite.java
* TokenizerExample.java

### Serialization

* SerializeExample.java
* DeserializeExample.java

### Exception Handling

* AccountUDE.java

### Socket Programming

* Client.java
* Server.java

## Concepts Covered

* Java Threads
* Synchronization
* User Defined Exceptions
* Byte Streams
* Character Streams
* Serialization & Deserialization
* String Tokenization
* Client-Server Communication
* Socket Programming

## How to Run

Compile:

```bash
javac FileName.java
```

Run:

```bash
java FileName
```
